ПОСТАВЛЕННЫЕ ПЕРЕД НАШИМИ ЭКСПЕРТАМИ ЗАДАЧИ:

 

  • Определить принципы премирования для всех уровней должностей Компании;
  • Разработать показатели премирования для каждой должности Компании;
  • Определить целевые значения показателей премирования для каждой должности Компании;
  • Определить порядок сбора данных для расчета и начисления премии сотрудникам всех уровней должностей Компании;
  • Разработать документы, регламентирующие процесс начисления и выплат премий сотрудникам Компании.

 

СОГЛАСОВАННЫЕ СРОКИ РЕАЛИЗАЦИИ ПРОЕКТА: 6 месяцев (включая 3 месяца внедрения)

 

ПОЛУЧЕННЫЙ РЕЗУЛЬТАТ ПРОЕКТА:

 

  • Определены принципы премирования для всех уровней должностей Компании;
  • Разработаны показатели премирования для каждой должности Компании;
  • Определены целевые значения показателей премирования для каждой должности Компании;
  • Определен порядок сбора данных для расчета и начисления премии сотрудникам всех уровней должностей Компании;
  • Разработаны документы, регламентирующие процесс начисления и выплат премий сотрудникам Компании.

 

ЦИФРЫ ПРОЕКТА:

 

  • Разработаны показатели премирования для 58 уникальных должностей;
  • Общее количество разработанных уникальных показателей премирования – 398 штук;
  • Передана технология разработки системы показателей премирования 15 линейным руководителям Компании.

 

ПОЛУЧЕННЫЙ ЭФФЕКТ ОТ ПРОЕКТА:

 

  • Оптимизация премиального фонда на 10%;
  • Повышение производительности труда сотрудников на 16%;
  • Повышение качества работы с клиентами, уровня клиентского сервиса;
  • Достижение поставленных перед компанией целей и задач.
