ПОСТАВЛЕННЫЕ ПЕРЕД НАШИМИ ЭКСПЕРТАМИ ЗАДАЧИ:

 

  1. Разработать модель профессиональных компетенций для ИТ-специалистов разного уровня;
  2. Разработать оценочные процедуры для оценки профессиональных компетенций ИТ-специалистов разного уровня;
  3. Подготовить инструкцию по предоставлению обратной связи сотруднику по итогам оценки;
  4. Передать технологию проведения оценки ответственным сотрудникам Компании;
  5. Подготовить рекомендации по внедрению модели профессиональных компетенций.

СОГЛАСОВАННЫЕ СРОКИ РЕАЛИЗАЦИИ ПРОЕКТА: 3 месяца

 

ПОЛУЧЕННЫЙ РЕЗУЛЬТАТ ПРОЕКТА:

 

  1. Разработана модель профессиональных компетенций для ИТ-специалистов разного уровня;
  2. Разработаны профили профессиональных компетенций для ИТ-специалистов разного уровня;
  3. Разработаны оценочные процедуры для оценки профессиональных компетенций ИТ-специалистов разного уровня;
  4. Подготовлена инструкция по предоставлению обратной связи сотруднику по итогам оценки;
  5. Проведен тренинг по передаче технологии проведения оценки профессиональных компетенций ответственным сотрудникам Компании;
  6. Подготовлены рекомендации по внедрению модели профессиональных компетенций;
  7. Подготовлены рекомендации по методам развития профессиональных компетенций специалистов разного уровня.

 

ЦИФРЫ ПРОЕКТА:

 

  • Количество разработанных моделей компетенций — 1
  • Количество разработанных профилей компетенций – 7
  • Количество разработанных комплектов оценочных процедур – 7
  • Количество проведенных тренингов по передаче технологии проведения оценки профессиональных компетенций – 1
  • Количество подготовленных методических инструкций – 1 (предоставление обратной связи по итогам оценки), 1 (методы развития профессиональных компетенций специалистов)

 

ПОЛУЧЕННЫЙ ЭФФЕКТ ОТ ПРОЕКТА:

 

  • Рост уровня компетенций сотрудников в среднем на 20% за 1 год
  • Повышение производительности труда сотрудников на 10%
  • Повышение качества работы с клиентами, уровня клиентского сервиса
